
# Employee Management System

## 📌 Project Description
The Employee Management System is a web-based application used to manage employee records such as employee details, department, and other information.

## 🚀 Features
- Add Employee
- Update Employee
- Delete Employee
- View Employee Details
- Department Selection

## 🛠 Technologies Used
- Java
- HTML
- CSS
- JavaScript
- Database (SQLite / MySQL)
- Git & GitHub

## 📂 Project Structure
Employee-Management-System
│
├── src
├── database
├── templates
├── static
└── README.md

## ⚙️ Installation
1. Clone the repository
2. Open project in VS Code
3. Run the Java application
